# Assembly FBLA

A Flutter project for the FBLA competition.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [Dart SDK](https://dart.dev/get-dart)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/amethysticality/assembly_fbla.git
   ```
2. Install packages
   ```sh
   flutter pub get
   ```

## Usage

To run the app, use the following command:

```sh
flutter run
```

## Building the project

To build the project for a specific platform, use the following commands:

### Android

```sh
flutter build apk
```

### iOS

```sh
flutter build ios
```

### Web

```sh
flutter build web
```

## Running Tests

To run the tests, use the following command:

```sh
flutter test
```
